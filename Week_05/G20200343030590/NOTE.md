# 动态规划

- 通过分解为子问题来简化复杂问题
Divide&Conquer + Optimal substructure
分治 + 最优子结构
关键点：
1. 动态规划和递归或者分治没有根本上的区别（关键看有无最优的子结构）
2. 共性：找到重复子问题
3. 差异性：最优子结构、中途可以淘汰次优解

- 状态转移方程：
opt[i,j] = opt[i+1, j] + opt[I, j+1]
if空地：opt[i,j] = opt[i+1, j] + opt[I, j+1]  else：0
关键点：
1. 最优子结构
2. 储存中间状态
3. 递推公式
